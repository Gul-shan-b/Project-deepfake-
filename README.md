# Text Deepfake Detection

A full-stack application for detecting AI-generated text using `roberta-base-openai-detector` from Hugging Face.

## Tech Stack
-   **Frontend:** React (Vite), Tailwind CSS, React Router, Axios
-   **Backend:** FastAPI, SQLite, SQLAlchemy, JWT Authentication
-   **AI/ML:** PyTorch, Transformers (`roberta-base-openai-detector`)

## Structure
-   `/backend` - Contains the Python FastAPI server.
-   `/frontend` - Contains the React Vite frontend application.
